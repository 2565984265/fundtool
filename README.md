# FundTool - 基金投资管理工具

一款纯前端、数据本地存储的基金投资管理工具。灵感来源于 [ifundtool.com](https://www.ifundtool.com/app/)。

## ✨ 功能特性

- 🚀 **快速添加基金**：输入基金代码自动获取基金信息
- 📊 **全局统计概览**：总成本、总市值、持仓收益、今日收益一目了然
- 📈 **持仓收益计算**：支持持仓收益、年化收益率 (TWR)、累计收益计算
- 💱 **交易记录管理**：买入、卖出、初始持仓记录，支持按时间顺序管理
- 📉 **估值趋势图表**：可视化展示基金估值走势
- 💾 **数据本地存储**：所有数据保存在浏览器 LocalStorage，保护隐私
- 🔄 **备份与恢复**：支持创建备份、导出 JSON、导入 JSON

## 🛠 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts（图表）
- Lucide React（图标）

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/2565984265/fundtool.git
cd fundtool

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📝 使用说明

1. 输入基金代码（如 `000001`）并设置初始持仓信息来添加基金
2. 系统会自动获取基金名称、净值等数据
3. 在基金详情中可以添加买入/卖出交易记录
4. 收益会自动根据交易记录重新计算
5. 所有数据保存在浏览器本地，可通过数据管理进行备份/导出/导入
